Tag HTML

Para que serve?

O par < html > e < /html > na gigantesca maioria dos casos, vem logo após a especificação do documento HTML, famoso , e geralmente envolve boa parte do código necessário para se criar um site. Essa tag diz ao navegador que o que estiver dentro dela deve ser encarado como elementos de um código HTML.

Tag BODY

Para que serve?

A tag < body > contém o corpo de um documento HTML, que é exibido pelo navegador em sua janela, ou seja, todo o conteúdo visível do site. É necessário que o < body > tenha ao menos um elemento "filho", ou seja, uma ou mais tags HTML dentro dele. Nesse exemplo, usamos a tag < h1 > , que indica o título principal da página.

Seletores CSS

O que são?

Um seletor CSS é a primeira parte de uma regra CSS. É um padrão de elementos e outros termos que informam ao navegador quais elementos HTML devem ser selecionados para que os valores de propriedade CSS dentro da regra sejam aplicados a eles.

Seletor por tag

O que é?

É o seletor mais abrangente da CSS, ele casa com todos os elementos encontrados.

Exemplo:h1, h2, p {color: red;}

Seletor por classe

O que é?

O seletor de classes CSS corresponde aos elementos com base no conteúdo de seus atributos class.

Este seletor básico escolhe elementos baseados no valor do seu atributo classe

Seletor por id

O que é?

Em um documento HTML, os seletores de ID do CSS selecionam um elemento baseado no conteúdo de seu atributo id, o qual deve ser exatamente igual ao valor dado ao seletor.SINTAXE:#valor_id { propriedades de estilo }
